# advlab
Youtube Dowload
โปรแกรม Python สำหรับดาวน์โหลดวิดีโอจาก YouTube ด้วย yt-dlp
ขั้นตอนที่ 1: ติดตั้งไลบรารีที่จำเป็น

ก่อนใช้งาน ต้องติดตั้ง yt-dlp ด้วยคำสั่ง

pip install yt_dlp


เพื่อให้ Python สามารถดาวน์โหลดวิดีโอได้

ขั้นตอนที่ 2: รันโปรแกรม

เปิด Command Prompt หรือ PowerShell แล้วรันไฟล์โปรแกรม เช่น

python dlyoutube.py

ขั้นตอนที่ 3: ผู้ใช้ใส่ URL วิดีโอ

โปรแกรมจะถาม URL ของวิดีโอ เช่น

https://youtu.be/tBiVYRHEo4g?si=GRA-DON0vzgdXcBl
